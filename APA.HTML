Tipos de dados

Na Linguagem de Programação Python, trabalharemos basicamente com 4 tipos de dados. São eles:
1 - Inteiro.
2 - Real.
3 - Texto.
4 - Lógico

Dados do tipo Inteiro

Os dados do tipo inteiro são aqueles pertecentes ao conjunto dos números inteiros, sem casas decimais. Na
programação,
utilizamos esse tipo de dado nos seguintes exemplos:

a - Quantidade de pessoas.
b - Quantidade de objetos em geral.
c - Quantidade de carros fabricados.

Dados do tipo Real

Os dados do tipo real são aqueles pertecentes ao conjunto dos números reais, com casas decimais. Na
programação,
utilizamos esse tipo de dado nos seguintes exemplos:

a - Quantidade de metros de uma parede.
b - Peso de uma pessoa.
c - Valor em dinheiro.

Dados do tipo Texto

Os dados do tipo texto são aqueles que representam qualquer texto manipulado no programa criado. Na
programação,
utilizamos esse tipo de dado nos seguintes exemplos:

a - Nomes de pessoas.
b - E-mail.
c - Endereço.

Dados do tipo Lógico

Os dados do tipo lógico são aqueles que podem ser utilizados para representar o estado das coisas. Na
programação,
esses valores são: Verdadeiro ou Falso. Observe alguns exemplos:

a - Se a luz estiver acesa o seu estado pode ser Verdadeiro, caso contrário será Falso..
b - Se uma porta estiver aberta o seu estado pode ser Verdadeiro, caso contrário será Falso.
c - Se a resposta para uma pergunta estiver certa o seu estado pode ser Verdadeiro, caso contrário será Falso.
